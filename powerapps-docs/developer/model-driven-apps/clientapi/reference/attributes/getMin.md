# <a name="getmax-client-api-reference"></a>getMax (クライアント API 参照)



属性で使用できる最小値を示す数を返します。 

## <a name="attribute-types-supported"></a>サポートされる属性の種類

少数、整数、倍精度浮動小数点、通貨

## <a name="syntax"></a>構文

`formContext.getAttribute(arg).getMin()`

## <a name="return-value"></a>戻り値

**種類**: 数値。 

**説明**: 属性の最小値。
