# <a name="getname-client-api-reference"></a>getName (クライアント API 参照)



属性の論理名を表す文字列を返します。 

## <a name="attribute-types-supported"></a>サポートされる属性の種類

すべて

## <a name="syntax"></a>構文

`formContext.getAttribute(arg).getName()`

## <a name="return-value"></a>戻り値

**種類**: 文字列。 

**説明**: 属性の論理名。
