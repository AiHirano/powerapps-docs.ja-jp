## <a name="syntax"></a>構文

`Xrm.Encoding.xmlEncode(arg)`

## <a name="parameters"></a>パラメーター

|パラメーター名        | 種類​​           | 必須出席者  |内容  |
| ------------- |-------------| -----|-----|
|引数        | String           | 必須出席者  |エンコードされる文字列。  |

**種類**: 文字列

**説明**: エンコードされた文字列。
