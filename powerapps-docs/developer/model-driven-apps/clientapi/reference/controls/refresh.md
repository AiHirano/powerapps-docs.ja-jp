# <a name="refresh-client-api-reference"></a>refresh (クライアント API 参照)



timelinewall およびタイマー コントロールに表示されているデータを最新の状態に更新します。

## <a name="control-types-supported"></a>サポートされているコントロールの種類

timelinewall、タイマー

## <a name="syntax"></a>構文

`formContext.getControl(arg).refresh();`
