# <a name="getlabel-client-api-reference"></a>getLabel (クライアント API 参照)



コントロールのラベルを返します。 

## <a name="control-types-supported"></a>サポートされているコントロールの種類

すべて

## <a name="syntax"></a>構文

`formContext.getControl(arg).getLabel();`

## <a name="return-value"></a>戻り値

**種類**: 文字列

**説明**: コントロールのラベル。
