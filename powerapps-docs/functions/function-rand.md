# <a name="rand-function-in-powerapps"></a>PowerApps の Rand 関数
疑似乱数を返します。

## <a name="description"></a>説明
**Rand** 関数は、0 以上 1 未満の疑似乱数を返します。

## <a name="syntax"></a>構文
**Rand**()
