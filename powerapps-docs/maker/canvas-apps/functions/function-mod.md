# <a name="mod-function-in-powerapps"></a>PowerApps の Mod 関数
除算の剰余を返します。

## <a name="description"></a>説明
**Mod** 関数は、数値を除数で除算した後に、その剰余を返します。

結果の符号は除数と同じです。

## <a name="syntax"></a>構文
**Mod**( *Number*, *Divisor* )

* *Number* - 必須。 除算される数値。
* *Divisor* - 必須。  除数。
