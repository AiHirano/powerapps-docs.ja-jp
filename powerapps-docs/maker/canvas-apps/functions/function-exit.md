# <a name="exit-function-in-powerapps"></a>PowerApps の Exit 関数
現在実行中のアプリを終了します。

## <a name="description"></a>説明
**Exit** 関数は、現在実行中のアプリを終了します。  アプリの一覧の表示に戻るので、ユーザーは別のアプリを選択して開くことができます。

## <a name="syntax"></a>構文
**Exit**()
